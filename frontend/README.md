# Fiesturo (React)

Regenerated React project for your Fiesturo site.

## Run

From `react-fiesta`:

```bash
npm install
npm run dev
```

## Routes

- `/` home
- `/ongoing`
- `/upcoming`
- `/past`
- `/registration`
- `/login`
- `/signup`
- `/contact`

## Assets

Put your images/videos in `react-fiesta/public/` (examples from your HTML):

- `a.jpeg`
- `background.mp4`
- `video1.mp4`, `video2.mp4`, `video3.mp4`
- `1.jpeg`, `2.jpeg`, `3.jpeg`, `4.jpeg`, `5.jpeg`, `7.jpeg`, `8.jpeg`
